# time-series-analysis

## Navodila za nastavitev okolja

### Priprava Python virtualnega okolja

1. **Ustvarite virtualno okolje:**
   ```bash
   python3 -m venv venv
   ```

2. **Aktivirajte virtualno okolje:**
   ```bash
   source venv/bin/activate
   ```

3. **Namestite potrebne pakete:**
   ```bash
   pip install -r requirements.txt
   ```
   
4. **Zaženite Jupyter Notebook:**
   ```bash
   jupyter notebook
   ```

### Deaktivacija virtualnega okolja

Ko končate z delom, deaktivirajte virtualno okolje:
```bash
deactivate
```
